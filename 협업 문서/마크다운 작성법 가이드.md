### :sparkles:코드
:exclamation: 백틱 (backtick, `) 사용
:exclamation: 첫 연속된 백틱 뒤에 해당하는 언어 입력

>\```(언어 이름) <br>
> (코드 내용)<br>
>  \```

#### ex) 
\```java <br>
int i = 0; <br>
int j = 10;<br>
\```

```java
int i = 0; 
int j = 10;
```
